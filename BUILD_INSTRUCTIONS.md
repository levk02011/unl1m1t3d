# 🔨 Инструкція по компіляції та розгортанню Minecraft Мода

## Що було додано:

### 1. **build_and_deploy.py** - Автоматизований скрипт
- Очищує попередні збірки
- Компілює мод через Gradle (`gradlew clean build`)
- Пошук готового JAR файлу
- Автоматичне копіювання в папку модів Minecraft

### 2. **build_mod.bat** - Швидкий запуск компіляції
Просто запустіть файл для:
```
Компіляція → Розгортання → Готовий мод
```

### 3. **build_and_play.bat** - Повний цикл
Запускає:
```
Компіляція → Розгортання → Запуск Лаунчера → Гра
```

### 4. **Кнопка в GUI лаунчері**
- Додана кнопка "🔨 Build Mod" перед кнопкою "Play"
- Клік на кнопку запускає компіляцію в окремому вікні
- Можна компілювати без закриття лаунчера

---

## 📋 Як використовувати:

### Варіант 1️⃣: Батник для компіляції (найпростіше)
```bash
# Двічі клікніть на:
build_mod.bat
```
Результат: Мод скомпільований і розгорнутий в папку модів

---

### Варіант 2️⃣: Компіляція + запуск гри в один клік
```bash
# Двічі клікніть на:
build_and_play.bat
```
Результат: Мод готовий, гра запущена

---

### Варіант 3️⃣: Через GUI лаунчер
1. Запустіть `qt_version.py` (або натисніть на лаунчер)
2. Натисніть кнопку **"🔨 Build Mod"**
3. Чекайте завершення компіляції
4. Натисніть **"Play"** для запуску гри

---

### Варіант 4️⃣: Python скрипт напряму
```bash
python build_and_deploy.py
```

## 📂 Структура файлів:

```

│   ├── build/               ← Компільовані файли
│   │   └── libs/
│   │       └── mod_1_21_4-1.0.0.jar  ← Готовий мод

    └── mods/                ← Папка для модів
```
