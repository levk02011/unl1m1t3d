# Тестування Minecraft мода

## Що зроблено:
- Мод скомпільовано успішно: `mod_1_21_4/build/libs/mod_1_21_4-1.0.0.jar`
- Створено Qt лаунчер для оффлайн гри без ліцензії
- Лаунчер автоматично встановлює мод для версії 1.21.4
- Гра запускається через Fabric без потреби в окремому лаунчері

## Як використовувати Qt лаунчер:

1. Запустіть `start.bat` - це відкриє Qt лаунчер
2. Введіть ім'я користувача (або залиште пустим для випадкового)
3. Виберіть версію "1.21.4 (з модом Fabric)"
4. Натисніть "Play"
5. Лаунчер завантажить Minecraft, встановить мод і запустить гру через Fabric

## Особливості:
- **Без ліцензії**: гра запускається в оффлайн режимі
- **Без окремого лаунчера**: все інтегровано в один Qt додаток  
- **Завжди з модом**: для 1.21.4 мод встановлюється автоматично
- **Через Qt**: сучасний графічний інтерфейс з індикаторами прогрессу

## Консольна версія:
Якщо потрібна консольна версія, використовуйте `console_version.py`

Мод готовий до тестування!
